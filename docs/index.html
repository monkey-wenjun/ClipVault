常见问题

关于 ClipVault 你可能想知道的问题

01

ClipVault 支持哪些平台？

ClipVault 支持 Windows 10/11、macOS（Intel/Apple Silicon）和 Linux（AppImage/deb/rpm），实现全平台覆盖。

02

数据同步是否安全？

是的，ClipVault 使用 AES-256 端到端加密，您的数据在离开设备前就已经加密，即使云盘提供商也无法读取您的数据。

03

支持哪些图床？

目前支持阿里云 OSS、七牛云、腾讯云 COS、AWS S3、GitHub、Gitee 等多种图床，后续会继续添加更多支持。

04

如何排除特定应用？

在偏好设置中，您可以添加需要排除的应用名称（如密码管理器），从这些应用复制的内容将不会被记录。
